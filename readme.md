# 📚 AI-Based Study Time Recommender System

This project is a Machine Learning-based system that recommends the optimal study time for students based on their daily habits.

## 🚀 Overview
This system predicts study time based on:
- Sleep hours
- Focus level
- Free time
- Days left before exams

## 🛠️ Technologies
Python, Pandas, NumPy, Scikit-learn, Streamlit, Joblib
